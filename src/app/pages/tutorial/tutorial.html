Bienvenue sur Collect'EAT !

L'application Collect'EAT permet de vous accompagner dans votre quotidien afin de faciliter l'accès à vos droits, nous allons ici vous présenter les différentes fonctionalités.

Faites glisser l'écran pour continuer

Menu principal

Le menu principal de l'application vous d'accéder rapidement à toutes les fonctionalités proposées

Pass collecte

Le bouton Pass collecte permet de générer un QR-code à présenter lors de la distribution des paniers repas afin de pouvoir retirer votre panier.

Trouver le centre de distribution

Le bouton Trouver le centre de distribution affiche une carte vous indiquant le centre de distribution où retirer votre panier repas.

Coffre-fort numérique

Le bouton Coffre-fort numérique permet d'accéder à votre espace de sotckage sécurisée où vous pouvez ajouter vos documents

Contacter votre assistante sociale

Besoin d'aide ? Le bouton Contacter votre assistante sociale permet de contacter votre assistante sociale pour vous assiter sur le fonctionnement de l'application ou pour tout autre question.

Merci d'utiliser Collect'EAT !
